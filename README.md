# sr-rendering

## Requirements

- Python 2.7
- RenderMan 23.5

## Usage

### Run

#### Render to default Framebuffer (it) directly

```shell
python ./app.py
```

#### Output to "rib file" and render

```shell
python ./app.py --rib
prman ./assets/candleholder.rib
```

### Command Line Help

```shell
python ./app.py --help
```
